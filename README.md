# ⚡ ChurnIQ - Customer Churn Prediction System

A machine learning web app that predicts customer churn and helps identify high-risk customers.

## 🚀 Features
- Login authentication system
- Role-based access (Pro users only)
- CSV upload for batch predictions
- Churn probability + risk segmentation
- Revenue at risk analytics
- Downloadable predictions

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
